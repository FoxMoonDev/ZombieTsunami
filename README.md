# 🧟 Zombie Tsunami Fan Clone

> **Projeto educacional** — desenvolvido para praticar HTML, CSS e JavaScript puro.

🎮 **[Jogar agora](https://foxmoondev.github.io/ZombieTsunami/)**

---

## Sobre o projeto

Este é um projeto **pessoal e não comercial**, criado com o único objetivo de estudar e praticar desenvolvimento web com tecnologias nativas do navegador — sem frameworks, sem bibliotecas externas, sem fins lucrativos.

O jogo foi construído do zero usando apenas:

- **HTML5** — estrutura e elementos da tela
- **CSS3** — visual, animações e layout responsivo
- **JavaScript (ES6+)** — lógica do jogo, física, colisão e controles
